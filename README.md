# Mini Jira

A modern Kanban task management app built with React, Tailwind CSS, and @dnd-kit.

---

## Features

- Add Task
- Edit Task
- Delete Task
- Drag and Drop
- Filters
- Undo Functionality
- LocalStorage Persistence

---

## Tech Stack

- React
- Tailwind CSS
- @dnd-kit
- Vite

---

## Installation

```bash
npm install
npm run dev
```

---

## Folder Structure

```txt
src/
├── components/
├── utils/
├── data/
└── App.jsx
```

---

## Drag and Drop

This project uses `@dnd-kit` for sortable drag-and-drop functionality.

---

## LocalStorage

Tasks are saved automatically in localStorage.
